Purpose: Single source of truth for preflight semantics across all surfaces. Once Wave 4 lands the MCP tool handlers, both protocols will call the same helper — policy and breaker behavior cannot drift between HTTP and MCP.
Output: Five routes refactored; the existing Phase 14 integration suite (`policy-gate-integration.test.ts`) passes unchanged; no behavioral diff at the HTTP wire level.
</objective>
