<objective>
Create the MCP host plugin shell — a Fastify plugin that registers a single multi-method route on `/mcp` (POST/GET/DELETE per the Streamable HTTP spec), maintains an in-process `Map<sessionId, SessionEntry>`, runs an idle-session GC sweep every `MCP_GC_INTERVAL_MS`, and tears down all sessions cleanly on app close with a 5-second hard ceiling. The plugin uses raw `req.raw`/`reply.raw` integration against `@modelcontextprotocol/sdk@^1.29.0`'s `StreamableHTTPServerTransport` (no community Fastify plugin — RESEARCH §Summary lock).
