     - SELECT the session row + verify agent_id matches + expires_at > now (throw `SessionAgentMismatchError` or `SessionExpiredError` or `SessionNotFoundError` appropriately).

     - Compute `newPending = computePendingToolCall(turn.role, turn.tool_calls, turn.tool_call_id, sessRow.has_pending_tool_call)` — see helper rule below.

     - Return `{ turn_id, turn_index, persisted: true }`.
   - Use `clearTimeout(timeoutHandle)` in the finally block.

3. `loadHistory(session_id, agent_id, opts?)`:
