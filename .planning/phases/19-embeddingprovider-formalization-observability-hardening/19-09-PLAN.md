**Post-ship gap closure for Phase 19 — DEPLOYMENT-ONLY.**

The Phase 19 RESS-WITH-TOOLS smoke gate fails (`DELTA_OK=0 COMPLETED_OK=1`) because the running `local-llms-router` Docker container is byte-stale: its `/app/dist/index.js` was bundled from a docker image dated **2026-06-01T15:42:09Z** — approximately 21 hours BEFORE the Plan 19-08 fix commit `aa4a9c6` (2026-06-02T12:21:18Z) which added the `delta.tool_calls[]` translation branch to `openAIChunksToCanonicalEvents`.

The source code is correct and unit-tested (4/4 cases in `router/src/translation/__tests__/openai-out.tool-call-streaming.test.ts` pass against current source). The deployed bundle is the only thing wrong. Because `compose.yml` declares `router: { build: ./router }` with no `image:` pin, plain `docker compose up -d` does not rebuild — it reuses the cached pre-fix image.

- Recreate the container against the fresh image

   ```bash
   run_ress_with_tools_gate() {
     local attempt="$1"
     local out
     out=$(mktemp /tmp/ress-tools-attempt-${attempt}-XXXXXX.txt)
     trap "rm -f '${out}'" RETURN

     curl -sS -N --max-time 60 \
       -X POST "http://127.0.0.1:3210/v1/responses" \
       -H "Authorization: Bearer ${ROUTER_BEARER_TOKEN}" \
       -H "Content-Type: application/json" \
       -d '{
         "stream": true,
         "model": "gpt-oss:20b-cloud",
         "input": "Call get_time to fetch the current UTC time.",
         "tools": [{
           "type": "function",
           "name": "get_time",
           "description": "Get the current UTC time in ISO-8601 format.",
           "parameters": {"type":"object","properties":{},"required":[]}
         }],
         "tool_choice": "required"
       }' > "${out}" 2>/dev/null || true

     local delta_ok=0 completed_ok=0
     grep -q "event: response.function_call_arguments.delta" "${out}" && delta_ok=1
     if grep -q "event: response.completed" "${out}" \
        && grep -q '"status":"incomplete"' "${out}" \
        && grep -q '"reason":"tool_calls"' "${out}"; then
       completed_ok=1
     fi

     echo "Attempt ${attempt}: DELTA_OK=${delta_ok} COMPLETED_OK=${completed_ok}"
     if [[ ${delta_ok} -eq 1 && ${completed_ok} -eq 1 ]]; then
       echo "PASS — attempt ${attempt}"
       # Stash the body for forensics if the operator wants it
       cp "${out}" "/tmp/ress-with-tools-PASS-attempt-${attempt}.txt"
       return 0
     fi
     # Print head for diagnostics
     echo "FAIL — attempt ${attempt}; body head (first 400 bytes):"
     head -c 400 "${out}"
     echo ""
     cp "${out}" "/tmp/ress-with-tools-FAIL-attempt-${attempt}.txt"
     return 1
   }
   ```
