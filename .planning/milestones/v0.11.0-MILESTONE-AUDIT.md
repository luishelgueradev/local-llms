## Tech Debt (non-blocking)

### Phase 16 — Test harness lesson
The original Phase 16 SHIP (2026-05-31) declared RESS-05 verified using a fake adapter that emitted canonical `tool_use` events directly, bypassing the real `openai-out.ts` translator. A real bug (missing `delta.tool_calls[]` branch in `openAIChunksToCanonicalEvents`) went undetected until Phase 19 introduced a live smoke gate against gpt-oss:20b-cloud. Closed by Plan 19-08 (source fix) and Plan 19-09 (Docker rebuild). Lesson recorded in `16-VERIFICATION.md` for the next milestone.

### Phase 17 — Test isolation hygiene
Two PG-gated test files have `session_id` prefix collisions when vitest runs them in parallel. Workaround: `--no-file-parallelism`. Not a Phase 17 implementation defect — it's pre-existing test infrastructure that could use polish.

### Phase 18 — Live tunnel deploy gap (operator action)
The Cloudflare tunnel at `https://local-llms.luishelguera.dev` is still serving the post-Phase-17 router binary. Phase 18 code is on disk and verified but not yet deployed to live traffic. Same root cause as the Plan 19-09 gap (`compose.yml` declares `build: ./router` with no `image:` pin, so plain `docker compose up -d` reuses the cached image). Recorded in `deferred-items.md`. Action: `docker compose up -d --build --force-recreate router && bash bin/smoke-test-router.sh --profile prod`. This is also surfaced as SEED-001 candidate for v0.12.0 Phase 1.
